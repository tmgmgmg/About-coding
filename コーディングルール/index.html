禁止事項

【EZgateコーディングルール】

http://ezcodingrules.ezdev.jp/?EZ CordingRules

  • 「./」禁止

    以下のように書く。

                  div class="p">○○○○</div>

  • ｐタグ禁止

    以下のように書く。

                  <div class="p">○○○○</div>

    ルール原本

  • table利用制限

    tableタグは、
    ・お問い合わせ
    ・会社内容
    ・採用情報
    といった指定の場所以外は使用しない。

  • CDN禁止

    jQueryなど使用の際はCDNではなく、ファイルを落としてきて、相対パスでリンクを貼る。

    ※googleフォント、住所自動入力のAjaxZip3は例外

    ルール原本

  • webフォント禁止

    指定のgoogleフォントのみ使用可。

    googleフォントにないフォントの指示があった場合は、画像としてpsdから切り出す。
    ※imgタグのalt属性には、切り出した文章を全て入力すること

  • 画像ファイル名の_(アンダーバー)禁止

    ―(ハイフン)は可。

  • Bootstrapのclass名に直接styleの指定禁止

    ・container
    ・row
    ・flex
    ・col-6
    などに直接styleの指定はNG。

  • IE読み込み時に、transitionが動作するバグへの対処

    以下コードを、各ファイルに記入

    ---HTML---
<body class="preload">

---CSS---
.preload * {
    -webkit-transition: none !important;
    -moz-transition: none !important;
    -ms-transition: none !important;
    -o-transition: none !important;
    transition: none !important;
}

---jQuery---
$(window).on('load' , function() {
  $("body").removeClass("preload");
});
